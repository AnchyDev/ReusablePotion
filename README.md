## About

Reusable Potion is a module for Azeroth Core that implements a reusable scalable potion in combat.

## Features

- Disable in PvP
- Heals a percentage.

## Install

- **[Clone](https://git-scm.com/docs/git-clone)** this repository into your `./azerothcore-wotlk/modules` folder.
- Re-run cmake.
- Re-recompile.
- You're done!
